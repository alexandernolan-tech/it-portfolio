# Home Lab Plan

## Short-Term Goals
- Create a Windows 10 VM in VirtualBox
- Practice snapshots and VM resets
- Break and fix VM networking
- Explore Windows services and dependencies

## Medium-Term Goals
- Install Windows Server 2022 (evaluation)
- Build first domain controller
- Join a Windows 10 client to the domain
- Practice creating users, groups, and permissions

## Long-Term Goals
- Try Group Policy Objects (GPO)
- Test DNS and DHCP roles
- Build a small multi-VM network
